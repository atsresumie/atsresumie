# atsresumiepro

A Next.js (App Router) web app that generates **ATS-optimized resumes**. Users provide three inputs — **Job Description**, **Current Resume**, and an optional **Prompt** — then the system outputs **LaTeX resume code** (ATS-friendly) that can be compiled into a **PDF** and delivered from a dashboard.

This repo is the **Next.js + Tailwind v4** version of the original Vite/React setup.

---

## Tech Stack

-   **Next.js 16** (App Router)
-   **React 19**
-   **Tailwind CSS v4** (CSS-first setup)
-   **shadcn/ui** + **Radix UI**
-   **TanStack React Query**
-   **React Hook Form** + **Zod**
-   **Sonner** (toasts) + shadcn Toaster
-   **Framer Motion**
-   Optional: `next-themes` for dark mode/theme switching

---

## Requirements

-   Node.js 18+ recommended
-   Package manager: pnpm / npm / yarn
